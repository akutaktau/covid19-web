Tentang Halaman Web Ini

Sumber data kes terbaru diperolehi daripada repositori Github ini. Data Rt dikira daripada kes aktif terbaru menggunakan kaedah Rolling Average dengan saiz data 14 hari. Data Kes per 100,000 penduduk dikira menggunakan data penduduk mengikut negeri yang diperolehi daripada halaman web DOSM.

Sebarang aduan / pertanyaan boleh diusulkan kepada saudara Jasdy Syarman di halaman Twitter.

Paparan terbaik pada skrin dengan resolusi 1920px × 1080px pada pelayan Microsoft Edge, Google Chrome atau Firefox.

Penafian

Pihak kami tidak bertanggungjawab terhadap sebarang kehilangan atau kerosakan yang dialami kerana menggunakan maklumat dalam portal ini.
